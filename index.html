My Resume

As a computer enthusiast and technology professional, I've spent more than 30 years enjoying technology's evolution from a niche interest to near-ubiquitous facet of daily life. Playing (and configuring) DOS games in the 80s led to programming in Pascal and working with Windows in the 90s, desktop support at the turn of the century and then network administration and software design in the 2000s.

I'm now taking the next step, from software design to development, with a focus on web applications. Core technologies that I'm learning include HTML, CSS and JavaScript. These are being built upon to include libraries and frameworks like LESS, SASS and React and backend technologies like Node, Java, Python and SQL.
